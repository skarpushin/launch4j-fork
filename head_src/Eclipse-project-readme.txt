In order to work with this project in Eclipse you need to make sure 
MinGW toolchain will be recognized by Eclipse. There are at least 2 options 
to achieve that:
	Option 1: Make sure to copy your MinGW distribution to C:\MinGW 
	Option 2: Specify environment variable MINGW_HOME to point to MinGW home, 
			  i.e. MINGW_HOME=c:\Dev-Cpp\MinGW64\
